# Glossary

- **Arcanum**: The application ecosystem built on ARCnet (modules + lived experience).
- **ARCnet**: The sovereign network / infrastructure substrate (Cosmos SDK–based chain providing settlement, identity anchoring, economy, and governance).
- **MANA**: ARCnet’s native utility token used for permissions, infrastructure usage, and governance participation (multi-factor).
- **ACC (Arcanum Chain Code)**: Identity anchoring mechanism binding participation to on-chain history (continuity, not description).
- **Hope**: Reflection interface module (structured inputs; no engagement-rank incentives).
- **Tempus**: Rhythmic time-window engine governing cadence and participation pacing.
- **Vitae**: Recognition framework for becoming (non-comparative; no worth scoring).
- **Nexus**: Social witness layer designed without virality mechanics or engagement-optimized ranking.
- **Rite**: A bounded unit of practice or contribution (often time-windowed by Tempus).
- **Treasury**: On-chain stewardship pool governed by proposals and bounded execution rules.
- **Governance**: Constitutional coordination system (bounded change; invariants protected).
- **Stability Guard**: Time-limited early-phase rollback/constraint authority used to preserve integrity during Genesis.
- **Doctrine Guard**: Integrity gate ensuring layer boundaries and dignity invariants remain intact.
- **IBC**: Inter-Blockchain Communication (Cosmos interoperability protocol).
- **De-listing**: Removing content from discovery surfaces while preserving addressability/referential integrity (when applicable).
