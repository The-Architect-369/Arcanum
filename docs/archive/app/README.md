# App Archive

This folder contains historical app-local documents that are no longer active implementation guidance.

The app may keep local docs for implementation details, but canonical behavior belongs in `docs/`, especially:

- `docs/architecture/`
- `docs/doctrine/`
- `docs/modules/`
- `docs/specs/app/`

Archived app docs may contain outdated routes, reward mechanics, gating assumptions, chain assumptions, or release-candidate language.
