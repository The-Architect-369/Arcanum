\# License \& Attribution



> \*\*Proposal — subject to community governance approval.\*\* Until ratified, treat this as the intended licensing posture.



\## 1. Documentation



\- \*\*Documentation (White Pages and docs):\*\* Creative Commons \*\*CC BY 4.0\*\* (attribution required)



\## 2. Code samples



\- \*\*Reference code samples (if included in docs):\*\* MIT License



\## 3. Creator content



\- \*\*Creator content published through Arcanum/ARCnet surfaces:\*\* creator-owned by default  

\- creators may optionally select alternate licenses per work (where the UI supports it)



\## 4. Attribution



Attribution text:



\- “Arcanum Documentation” and/or “Arcanum White Pages”

\- Include a link to the canonical repository or release page
